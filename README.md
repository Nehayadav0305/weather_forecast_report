# weather_forecast_report
A modern, interactive **Power BI dashboard** that provides real-time weather information and air quality insights for Jaipur, Rajasthan.  Built with a clean dark theme and user-friendly layout, this dashboard helps users quickly understand current weather conditions, weekly forecasts, and pollution levels.
